# To Start

**Clone the project**  

```Bash
git clone https://github.com/ShengyuanWang/COMP225-Project.git
```

**Download the packages**

```Bash
pip install -r requirements.txt
```

**Start Backend Env**

```Bash
cd backend
flask run
```

**Start Frontend Env**

```Bash
npm init vue@latest
cd frontend
npm install
npm run dev
```
